Проект открывать с помощью среды разработки IntelliJ IDEA.

Как открыть?
1.Разархивировать файл, в любое место (главное запомните это место)
2.Заходим в IntelliJ IDEA
3.Нажимаем кнопку File
4.Дальше Open
5.Потом идёт туда где хранится ваш файл, например, если он находится в загрузках: C:\Users\Сергей\Downloads\LB4
6.Теперь когда мы дошли до папки где хранятся файлы проекта, нажимаем кнопку OK
